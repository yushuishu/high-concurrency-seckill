## 使用

1、创建数据库，修改配置文件`application.yml`的数据库连接

2、在项目根目录下，执行SQL文件`database-sql.sql` 文件中的SQL语句，创建表，并插入商品数据。

3、启动项目，访问接口文档（接口已排序）

4、参考着代码，进行测试
